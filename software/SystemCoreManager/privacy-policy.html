Privacy Policy

Accessible Tech Labs India is committed to ensuring complete privacy, transparency, and user control. This software is designed as an offline-first system utility, meaning that almost all features operate locally on your device without sending any information to external servers. We do not collect, store, share, or transmit personal data. All operations are executed securely within your own system environment.

Run History Maintenance

The Run History Maintenance module reads and manages the Run dialog history stored in the Windows registry. This includes viewing existing entries, counting them, or clearing them completely. All actions are performed locally using system-level access. No history data is stored by the application or transmitted externally. The user has full control over whether to view or delete this data.

Scheduled Cleanup System

This feature allows users to automate the cleaning of Run history using Windows Task Scheduler. When enabled, a scheduled task is created within your system that runs at a specified time and performs cleanup operations. The application does not monitor or track this activity. All automation is handled directly by Windows itself, ensuring reliability and privacy.

Wi-Fi Credential Manager

The Wi-Fi Credential Manager retrieves saved Wi-Fi profiles and passwords using secure system commands. This includes viewing the currently connected network, listing all saved networks, and exporting credentials into a local file if requested. All data remains on your device and is never stored, logged, or transmitted by the application. The export feature simply creates a file chosen by the user, giving full control over where the information is stored.

Automatic Power Manager

This module enables users to schedule automatic system actions such as shutdown, restart, and sleep. It uses Windows Task Scheduler to create and manage tasks. The application does not collect or store any information related to system usage or behavior. All schedules are created only when initiated by the user and can be enabled, disabled, or deleted at any time.

Power Schedule Management

Users can view, enable, disable, or permanently delete scheduled power tasks. The application simply interacts with existing system tasks and displays their status. No background monitoring or hidden processes are involved. Everything is fully transparent and controlled by the user.

USB Security Management

The USB Security Management feature provides advanced control over USB storage access to enhance system protection and prevent unauthorized data transfer. Users can enable or disable USB storage functionality with a single action, ensuring complete control over external storage devices. When USB protection is enabled, all USB storage devices such as pen drives and external hard disks are completely blocked from the system. These devices will not appear in File Explorer and cannot be used for reading or transferring data. This helps prevent data theft, malware transfer, and unauthorized access. The feature is designed with safety in mind, ensuring that essential input devices such as USB keyboards and mice continue to function normally at all times. The system modification is fully reversible, allowing users to restore USB access instantly whenever required. Additionally, users can check the current USB status at any time to determine whether USB storage is enabled or disabled. All operations are performed locally on the system without any external communication, ensuring complete privacy and security. This feature is especially useful in shared environments, educational setups, and security-focused systems where controlled access to external storage is required.

Settings and Update System

The Settings module provides options such as checking for updates and viewing the privacy policy online. The application may connect to the internet only when the user manually chooses to check for updates or open a web page. No personal data, system data, or identifiers are sent during this process. Update checks only retrieve version information.

Data Security and Safety

All features are designed to operate safely within the Windows environment without causing system damage. The application does not modify critical system components beyond necessary user-requested actions. No hidden changes, tracking mechanisms, or background data collection are present. Security and stability are prioritized in every feature.

User Control and Transparency

The user is always in full control of all operations. Every action requires user interaction, and nothing runs automatically without permission. There are no hidden features or background activities. All processes are visible, understandable, and reversible.

Offline Functionality

This software is primarily designed to work offline. Except for optional update checks or opening web links, all features function without internet access. This ensures maximum privacy and independence for the user.
