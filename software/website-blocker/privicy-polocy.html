Privacy Policy

Accessible Tech Labs India respects your privacy and is committed to protecting it. This Privacy Policy explains how our Website Blocker application operates and how user information is handled.

1. Information Collection

The Website Blocker application does not collect, store, transmit, or share any personal information. The software operates completely offline and performs all actions locally on the user’s device.

2. Data Usage

No user data, browsing history, personal identifiers, or system information is collected by this application. All website blocking and unblocking actions are performed only by modifying the local system configuration with the user’s permission.

3. Internet Connectivity

The application does not connect to any external servers. If an Internet Lock Mode is enabled, it is done strictly at the system level and remains fully under the user’s control.

4. Third-Party Services

This application does not use or integrate any third-party analytics, tracking tools, advertisements, or external services.

5. Accessibility Commitment

Accessible Tech Labs India designs software specifically with accessibility in mind. This application is optimized for screen readers and is suitable for blind and visually impaired users.

6. User Control

Users have full control over all features of the application, including enabling or disabling website blocking and internet access. All changes can be reversed at any time by the user.

7. Policy Updates

This Privacy Policy may be updated from time to time to reflect improvements or feature changes. Any updates will maintain the same commitment to user privacy and security.

8. Contact Information

If you have any questions or concerns regarding this Privacy Policy or the application, you may visit our official website for more information.
