Desktop Software Tools

This section provides downloadable desktop utilities developed under Accessible Tech Labs India. Each application is designed with accessibility first — supporting screen readers, full keyboard control, and simple command-line workflows. All tools run completely offline after download and are built especially for blind users and beginners who prefer clean and reliable utilities.

System Diagnosis Engine

A professional Windows command-line diagnostics and reporting tool created specifically for NVDA and blind users. It generates structured system reports including hardware information, operating system details, network configuration, storage status, and performance data. The tool provides spoken-friendly output, progress feedback, categorized menus, and clear instructions, making it ideal for troubleshooting, system auditing, and collecting technical information in an accessible and professional way.

Audio Video Converter

A lightweight command-line media conversion utility that allows users to convert audio files into video and extract audio from video files. When converting audio to video, users can choose from multiple professional background colours or provide their own image. Supported formats include MP3, WAV, and MP4. The entire workflow is keyboard driven with clear prompts at every step, making it perfect for blind users who need simple media conversion without complex graphical interfaces.

YouTube Downloader

An accessible command-line downloader built on offline engines that enables users to download audio, video, playlists, and full channels directly to their computer. It supports MP3, M4A, WAV, and MP4 formats with automatic folder organization. The menu-based interface is designed for screen readers and keyboard navigation. This tool is intended for educational and personal use only. Users should always respect platform terms and copyright laws when downloading content.
