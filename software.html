Desktop Software Tools

This section contains downloadable desktop utilities developed under Accessible Tech Labs India. All tools are designed for keyboard navigation, screen reader compatibility, and simple command-line workflows. These applications run fully offline after download.

System Diagnosis Engine

A professional Windows command-line system reporting tool created specifically for blind and NVDA users. This utility generates multiple categorized reports including hardware information, operating system details, network configuration, storage status, and performance data. The tool uses structured menus, spoken feedback compatibility, and clear progress messages. It is ideal for troubleshooting, system auditing, and collecting technical information in an accessible way without requiring advanced technical knowledge.

Download System Diagnosis Engine

Audio Video Converter

A lightweight command-line utility that converts audio files into video and extracts audio from video files. When converting audio to video, users can choose from multiple professional background colours or provide their own image. Supported formats include MP3, WAV, and MP4. The tool is fully keyboard driven and provides clear prompts at every step, making it suitable for blind users who want simple media conversion without complex graphical interfaces.

YouTube Downloader

An accessible command-line downloader built on offline engines that allows users to download audio, video, playlists, and full channels directly to their computer. It supports MP3, M4A, WAV, and MP4 formats with automatic folder organization. The menu-driven interface is designed for screen readers and keyboard navigation. This tool is intended for educational and personal use. Users should respect platform terms and copyright laws when downloading content.
